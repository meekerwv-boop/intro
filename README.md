# Hi, I'm John Meeker 👋

Program coordinator and nonprofit leader turned data analyst —
based in Shepherdstown, WV, rooted in Appalachia.

I spent a decade building systems that help mission-driven
organizations thrive: managing operations, leading teams, and
doing the unglamorous work that keeps community programs running.
Then I went and learned to make the data talk too.

I recently completed an intensive data analytics bootcamp where
I built dashboards, wrangled messy datasets, mapped community
challenges with geospatial tools, and connected it all back to
the work that actually matters — food security, community health,
and regional resilience.

I bridge the gap between nonprofit operations and data. I
understand both the spreadsheet and the story behind it.
